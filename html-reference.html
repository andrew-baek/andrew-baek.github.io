Reference Page

  • <html>

    • This tag tells the browser that this is an HTML document and represents the root of an HTML document. The <html> tag is the container for all other HTML elements (except for the <!DOCTYPE> tag).

  • <head>

    • This tag is a container for all the head elements and can include a title for the document, scripts, styles, meta information, and more.

  • <title>

    • This tag is required in all HTML documents and it defines the title of the document. The tag defines a title in the browser toolbar, provides a title for the page when it is added to favorites, displays a title for the page in search-engine results

  • <body>

    • This tag defines the document's body and contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

  • <h1>-<h6>

    • These tags are used to define HTML headings. <h1> defines the most important heading. <h6> defines the least important heading.

  • <p>

    • This tag defines a paragraph. Browsers automatically add some space before and after each element.

  • <br>

    • This tag inserts a single line break and is an empty tag which means that it has no end tag.

  • <hr>

    • This tag defines a thematic break in an HTML page and is used to separate content (or define a change) in an HTML page.

  • <!-- -->

    • This tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.

  • <a>

    • This tag defines a hyperlink, which is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

  • <img>

    • This tag defines an image in an HTML page and has two required attributes: src and alt.

  • <ul>, <ol>, <li>

    • These tags defines a list item. The <li> tag is used in ordered lists(<ol>), unordered lists (<ul>), and in menu lists (<menu>).

  • <table>, <tr>, <th>, <td>

    • These tags defines an HTML table. An HTML table consists of the <table> element and one or more <tr>, <th>, and <td> elements. The <tr> element defines a table row, the <th> element defines a table header, and the <td> element defines a table cell.

  • <div>

    • This tag defines a division or a section in an HTML document and is used to group block-elements to format them with CSS.
